For #5–7 you can trigger applyDiffAsSuggestion directly from the browser console if you expose the editor view:

```javascript
applyDiffAsSuggestion(view.state, view.dispatch, {
	nodeId: "...", // copy from DOM: data-node-id attribute
	proposedText: "...",
	authorID: "AI Assistant",
});
```
