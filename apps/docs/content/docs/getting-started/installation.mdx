## Prerequisites

- **Node.js** ≥ 18 (LTS recommended)
- A package manager — `pnpm`, `npm`, or `yarn`
- A **React** application (React 18 or 19)

<Callout type="info">
  Inscribe is written in strict TypeScript. You don't need to install separate `@types/*` packages — type declarations are bundled with each package.
</Callout>

---

## Install the Packages

For a standard React application install the core engine and the React bindings:

<Tabs items={["pnpm", "npm", "yarn"]}>
  <Tab value="pnpm">
    ```bash
    pnpm add @inscribe/core @inscribe/react
    ```
  </Tab>
  <Tab value="npm">
    ```bash
    npm install @inscribe/core @inscribe/react
    ```
  </Tab>
  <Tab value="yarn">
    ```bash
    yarn add @inscribe/core @inscribe/react
    ```
  </Tab>
</Tabs>

### Optional Packages

| Package | Purpose & Features |
| --- | --- |
| `@inscribe/plugins` | **Real-time collaboration** (Yjs/Hocuspocus), **AI Toolkit**, and **Track Changes**. |
| `@inscribe/export` | Exporting documents to paginated **PDF** (via `pdf-lib`), **DOCX** (via OpenXML), and **Markdown**. |

---

## Setting Up Optional Features

If you opt into the features provided by `@inscribe/plugins`, you will need to configure the corresponding external dependencies and environment variables in your application.

### AI Toolkit (Vercel AI SDK)

To use the AI writing assistant (powered by Anthropic Claude by default), you will need to install the Vercel AI SDK and set up a server-side endpoint:

<Tabs items={["pnpm", "npm", "yarn"]}>
  <Tab value="pnpm">
    ```bash
    pnpm add ai @ai-sdk/react @ai-sdk/anthropic
    ```
  </Tab>
  <Tab value="npm">
    ```bash
    npm install ai @ai-sdk/react @ai-sdk/anthropic
    ```
  </Tab>
</Tabs>

**Environment Variables:**
*   `ANTHROPIC_API_KEY`: Required by your backend API route to fulfill AI streaming requests.

### Real-Time Collaboration (Hocuspocus/Yjs)

Inscribe's collaborative engine relies on **Yjs** for CRDT data structures and synchronizes over WebSockets via **Hocuspocus**. You will need a running Hocuspocus backend server to persist and sync editor states. 

**Environment Variables:**
*   `VITE_WS_URL`: The frontend client needs the address of the collaboration server (e.g., `ws://localhost:1234`).

## Bundler Support

Inscribe has been tested with **Vite** (recommended), **Webpack 5**, and **esbuild**. No special plugins or aliases are required — the packages ship pre-bundled ESM.
